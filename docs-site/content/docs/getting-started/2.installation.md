# Installation

## Requirements

- Python **3.11** or higher
- `pip` / `pipx` / [Poetry](https://python-poetry.org/) 1.8+

## From PyPI

```bash
pip install upki-ca
```

Or with [pipx](https://pipx.pypa.io/) to keep it isolated:

```bash
pipx install upki-ca
```

## From source (development)

```bash
git clone https://github.com/circle-rd/upki-ca.git
cd upki-ca

# Install with Poetry
poetry install

# Or with pip in editable mode
pip install -e ".[dev]"
```

## Docker

```bash
docker pull ghcr.io/circle-rd/upki-ca:latest
```

Or build locally:

```bash
docker build -t upki-ca .
```

## Verify installation

```bash
upki-ca --version
```

Expected output:

```
upki-ca, version X.Y.Z
```
